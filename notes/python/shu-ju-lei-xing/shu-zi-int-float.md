# 数字(int, float)

**常用方法：**

*   **数学运算**：

    ```python
    abs(x)                  # 返回绝对值
    round(x, n)             # 四舍五入到 n 位小数
    pow(x, y)               # 返回 x 的 y 次方
    divmod(x, y)            # 返回商和余数
    ```
*   **类型转换**：

    ```python
    int(x)                  # 转换为整数
    float(x)                # 转换为浮点数
    ```
*   **其他**：

    ```python
    max(x, y)               # 返回最大值
    min(x, y)               # 返回最小值
    ```
